# Yelp Clone (Full Stack)

A yelp clone web application built in NodeJS and MongoDB where users can search, geolocate, review and curate their favourite restaurants from around the world.

## Getting Started

Make sure you have the latest version of Node.js installed. If you don't have it, or are an older version, you can visit Nodejs.org and download the installer.


### Installing

A step by step series of examples that tell you have to get a development environment running

Intital Install:

```
npm install
```

Running the program locally:
```
npm start
```
